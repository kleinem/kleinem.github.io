Project Reflection :

Being the first digital team game I worked on, the development of this game was the first in which source control and asset management needed to be used to coordinate between several developers. It was also one of the first where I needed to create a tech demonstration of the concept. This would prove to be an invaluable skill in future projects. I came out of this project with a much better understanding of how teams function, and the typical production process from start to finish.
